# Custom-32-Bit-CPU
A custom CPU architecture made from scratch in Logisim and later made in Verilog to be embedded into FPGA systems. The CPU contains a total of:

 - [8] 32-bit Registers (Logisim) or [16] 32-bit Registers (Verilog)
 - Custom Instructions
 - ALU
 - PC counter
 - MAR and MDA

For more information about the two implementation click on the directory above to get a full information in detail and how to use it yourself.
